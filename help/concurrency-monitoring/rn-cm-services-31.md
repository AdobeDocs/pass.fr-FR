# Notes De Mise À Jour D’Adobe Concurrency Monitoring Services 3.1 {#cm-services-rns}

Cette page décrit les nouvelles fonctionnalités, les modifications et les problèmes connus de cette version.

## Informations sur la version {#release-info}

Version : 3.1

Date de publication : 09/03/2020

## Présentation de la version {#release-overview}

* Nouveau rapport sur le niveau de simultanéité. Ce rapport reflète le nombre de flux actifs simultanés pour le même compte, dans une fenêtre temporelle. Un flux actif est défini comme étant un flux qui a été démarré avec succès.
* Nouveau rapport sur le niveau d’activité. Ce rapport reflète toute activité de l’utilisateur, dans une fenêtre temporelle : flux actifs et tentatives d’initialisation de flux refusées après l’application de la politique.
* Cette version s’accompagne d’améliorations internes qui simplifieront le processus de publication de nouveaux rapports à l’avenir.
* Nouvelle interface utilisateur de gestion de la configuration interne qui utilise l’API REST Configuration interne lancée en mai 2020.
* Divers correctifs de bugs et améliorations de l’API de configuration REST afin de faciliter la nouvelle interface utilisateur de gestion de la configuration.

## Problèmes connus

Aucune.
